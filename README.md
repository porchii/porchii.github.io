Множества - набор из обьектов, однозначно определяемый своими составляющими.
Множества можно порождать путем перечисления каких-либо обьектов, либо указать свойства, которыми эти обьекты обладают.

P(x) - некоторое свойства, будем задавать множества обладащий этим свойством так: $X =\left\{ x: p(x)) \right\}$
